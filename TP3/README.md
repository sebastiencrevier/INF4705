- Si le programme `tp3` ne s'exécute pas sous Linux, faire la commande `chmod u+x tp3`,
puisque le transfert du programme de Linux à Windows retire parfois les droits en lecture.
- Si cela ne fonctionne pas, simplement aller dans le répertoire `LaurentidesTrails` et appeler `make` et déplacer `tp3` à la racine.
